# 📊 Retail Sales & Inventory Analytics

An interactive Power BI dashboard built to analyze retail sales performance, inventory distribution, and outlet-level business insights using Blinkit grocery data.

## 🚀 Project Overview

This project focuses on analyzing retail sales data to identify business trends, product performance, outlet efficiency, and customer ratings. The dashboard enables decision-makers to monitor key performance indicators (KPIs) and make data-driven business decisions.

## 🛠️ Tech Stack

- Power BI
- DAX
- Excel

## 📂 Dataset

The dataset contains retail grocery sales information including:

- Product Categories
- Sales
- Outlet Type
- Outlet Size
- Outlet Location
- Item Visibility
- Customer Rating
- Fat Content
- Establishment Year

- Identified top-performing outlet locations based on total sales.
- Compared sales performance across different outlet sizes.
- Analyzed product category contribution to overall revenue.
- Evaluated customer ratings and item visibility trends.
- Monitored outlet establishment trends over time.
